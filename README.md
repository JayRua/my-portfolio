# My Portfolio Website

This is my personal portfolio site showcasing my background in tech sales, marketing and computing, along with selected projects.  

## 📖 About Me

With a Master’s in Digital Marketing Strategy from Trinity College Dublin and six years’ experience across digital marketing, tech sales, and account leadership, I’ve built a career at the intersection of sales, commercial growth, technology and marketing.

At Pinterest via Accenture Ireland, I worked with some of the world’s most recognisable brands to deliver high-impact campaigns, drive revenue growth, and lead top-performing sales teams. My work combined strategic marketing with solution-based sales, supported by technical skills in SQL, VBA, and workflow optimisation. One of my proudest moments was launching a brand-new tech sales team for Pinterest in Kuala Lumpur, built from the ground up using the lessons learned at our Dublin HQ.

Now expanding my technical expertise with a Higher Diploma in Science in Computing at ATU Donegal, I’m focused on leveraging my skillset in sales, marketing, AI and technical know-how to deliver measurable results for businesses

## Folder Structure
- `index.html` – Homepage with About, Work, and Contact sections
- `assets/` – CSS, JavaScript, images
- `projects/` – Individual project detail pages
- `README.md` – This file

## Viewing Locally
1. Download or clone this repository
2. Open `index.html` in your web browser

## Deployment
This site is hosted via GitHub Pages.  
To publish:
1. Push changes to the `main` branch
2. In GitHub, go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. Save and use the provided link
